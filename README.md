# Linho1219/github-profile-summary-cards

Yet another github profile summary card generator.

This is a fork of [vn7n24fzkq/github-profile-summary-cards](https://github.com/vn7n24fzkq/github-profile-summary-cards) and I changed the following things:

- Use pnpm instead of npm
- Use Vite instead of ncc
- Use Vitest instead of Jest
- Remove Vercel APIs
- Add "others" section to language chart
- Add theme filter to generate only some of the themes

I use this fork in my own profile page. Check it out at [Linho1219/Linho1219](https://github.com/Linho1219/Linho1219).
